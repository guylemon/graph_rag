## Task
Extract entities and relationships from the input text.

## Rules:
1) Extract all entities whose type is allowed in the schema.
2) For each entity, return:
   - entity_name (uppercase canonical name)
   - entity_type (must be one of allowed types)
   - entity_description (concise but informative)
3) Return only JSON matching the provided schema.

## EntityType reference:
- CONCEPT: Theoretical ideas, methodologies, approaches
- EVENT: Conferences, releases, historical events
- LIFEFORM: Non-human lifeforms like plants, animals, insects
- LOCATION: Cities, countries, regions
- ORGANIZATION: Companies, institutions, or universities
- PERSON: People mentioned in the chunk text
- PRODUCT: Software products, platforms, services
- TECHNOLOGY: Frameworks, programming languages, algorithms
